🌦️ Real-Time Weather Analytics Dashboard

A modern end-to-end Weather Analytics project built using Python, MySQL, SQL, and Power BI.
This dashboard fetches real-time weather data using the OpenWeather API, stores it in MySQL, performs analytical queries using SQL, and visualizes insights through an interactive Power BI dashboard.

🚀 Project Overview

This project automates the complete weather analytics workflow:

✅ Fetch real-time weather data from API
✅ Store cleaned data into MySQL database
✅ Prevent duplicate records
✅ Perform SQL-based analysis
✅ Create dynamic Power BI dashboard
✅ Generate interactive KPI insights
